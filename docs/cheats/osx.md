# Cheats - OSX

## OSX

**Ctrl+Up**
: Mission Control / space management

**Ctrl-right/left**
: Move to other spaces

**Ctrl+Cmd+click+drag**
: Move Window

**Ctrl+Cmd+space**
: Open special character window

**Cmd+Tab**
: Switch windows

**Cmd+h**
: Hide window

**Cmd+c**
: Copy

**Cmd+v**
: Paste

**Cmd+z**
: Undo

**Cmd+Shift+.** - in finder window
: Show hidden files
