# Kitty Cheats

## Kitty

### Config

|              |               |
| ------------ | ------------- |
| `Ctrl+Cmd+,` | Reload config |

### Windows

|                   |                        |
| --                | --                     |
| `Cmd+enter`       | New window             |
| `Cmd+t`           | New tab                |
| `Cmd+w`           | Close tag              |
| `Ctrl+Cmd+arrows` | Resize window          |
| `Shift+Cmd+d`     | Close window           |
| `Alt+Cmd+t`       | Change window title    |
| `Cmd+[]`          | Previous / next window |

### Help

|          |                   |
| --       | --                |
| `Cmd+F6` | Show kitty config |
